# Maison Soleil - Booking Confirmation Page

A beautiful, highly-polished, and fully responsive hotel booking confirmation dashboard built with vanilla HTML, CSS, and JavaScript. This project replicates a premium boutique hotel guest portal featuring interactive UI elements and a warm, elegant aesthetic.

## ✨ Features

- **Fully Responsive Layout:** Adapts seamlessly from desktop to tablet and mobile screens.
- **Interactive "Fan" Effect:** Hovering over the hero section elegantly fans out the Receipt and Welcome cards.
- **Clipboard Integration:** One-click button to copy the Wi-Fi password to the user's clipboard with visual feedback.
- **Custom Typography:** Uses a blend of 'Inter' for UI text and 'Playfair Display' for elegant headings.
- **Pure CSS Graphics:** The receipt barcode and sun icons are generated using CSS gradients and inline SVGs, requiring no external image assets.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, and CSS Transitions/Animations.
- **Vanilla JavaScript:** DOM manipulation and Clipboard API.
